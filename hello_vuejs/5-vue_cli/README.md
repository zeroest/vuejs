
# [Vue CLI](https://cli.vuejs.org/)

## Getting Started

### Install:

```
npm install -g @vue/cli
(npm install --global @vue/cli)
# OR
yarn global add @vue/cli
```

### Create a project:

```
vue create my-project
# OR
vue ui
```
