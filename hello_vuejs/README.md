## 컴포넌트 옵션 속성

아래 명칭은 모두 같은 의미
- 컴포넌트 옵션 속성
- 인스턴스 옵션 속성
- 옵션 API

Vue 생성시 입력되는 옵션을 의미한다

```vue
new Vue({
    // 하위 내용을 컴포넌트 옵션 속성이라 부른다 
    el: ,
    template: ,
    data: ,
    methods: ,
    created; ,
    watch: ,
})
```
